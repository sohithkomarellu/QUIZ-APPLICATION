# QUIZ-APPLICATION

An Online Quiz Application developed using **Spring Boot** for backend and **HTML, CSS, JavaScript** for frontend.  
This application allows teachers to create and manage quizzes, and students to attempt quizzes using a Quiz ID.

---

## Features

### Teacher Panel
- Add new questions
- View all questions
- Filter questions by category
- Delete questions
- Create quizzes

### Student Panel
- Enter Quiz ID to attempt quiz
- Interactive quiz interface
- Quiz timer
- Automatic score calculation

---

## Technologies Used

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- REST APIs
- MySQL Database

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)
